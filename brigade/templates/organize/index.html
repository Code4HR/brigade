Resources

Organizing a Brigade? We've got you covered.

Over the past five years, we’ve learned a lot about what works when organizing a Brigade. We’re excited to help you build a sustainable, civic hacking group in your community.

Here are some resources to help you get started:

Organizer’s Playbook

The Playbook gives practical advice and real examples on starting and running a sustainable Brigade.

It includes: guidance on hosting hack nights, building a core team, and practicing user-centered design.

View Playbook

Organizer’s Handbook

The Handbook points to the resources that CfA offers Brigades and how to take advantage of them.

It includes: how to get set up on Meetup, how to use Expensify, and information about the National Advisory Council (NAC).

View Handbook

Software Available to Brigades

Learn about the software that vendors have donated for Brigades to use.

It includes: cloud hosting, communication, mapping, and other software.

Explore Software
