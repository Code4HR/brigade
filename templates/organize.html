Want to organize a Brigade? We've got you covered.

Over the past four years, we’ve learned a lot about what works when organizing a Brigade. We’ve compiled a playbook, calendar, and tools for Brigade organizers to use to help build a sustainable, civic hacking group in your community.

Code for America has a few requirements for groups that want to join the Brigade network, either as a civic hack night or official Brigade. You can use the Brigade Organizer's Playbook to help get these pieces in place. Once you're all set, let us know. You'll then have access to the range of support Code for America offers it's Brigades.
